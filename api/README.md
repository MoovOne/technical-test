This is an API using [json-server](https://github.com/typicode/json-server). 

## Getting Started

First, run the development server:

```bash
yarn dev
```

This will start an API on [http://localhost:3001](http://localhost:3001) with the following routes:

```
GET    /todos
GET    /todos/1
POST   /todos
PUT    /todos/1
PATCH  /todos/1
DELETE /todos/1
```
